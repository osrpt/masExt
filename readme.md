##masExt
masExt是一个提供更好的网页浏览体验的chrome扩展工具。

##主要功能：
1. 整合微博急简css，在此基础上做了一些修改
2. 修改QQ空间v8样式
3. 屏蔽zhibo8广告，筛选出NBA的比赛，高亮热火队的比赛
4. 修改博客园的样式
5. 网页内翻译功能
6. 添加对上微博的提醒
